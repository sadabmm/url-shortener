SMM URL Shortener


Type in any valid URL and submit it. You will receive a shortened version of your URL. Share it with everyone!


Example Usage:

  • Submit any URL in the format "http://www.example.com" and get a JSON response with a shortened URL.

    {
    "original": "http://www.example.com",
    "shortened": "https://smm-url.glitch.me/16983"
    }

  • Visit the shortened URL and get redirected to the original URL anytime! Goto "https://smm-url.glitch.me/16983" and get redirected to "http://www.example.com".
